文件目录：

#ifndef ... #define ...

上述两段作用相同，都是为了防止该头文件被多次包含。

_HEAD_H是头文件的标识符，我们约定全部大写。

HEAD是其文件名，H表示.h文件（头文件）。

具体逻辑是：如果该头文件从未被包含，那么就包含该文件，即#define,定义_HEAD_H表示该文件被包含。

如果下一次又要包含该文件，因为_HEAD_H已经被定义，表明该头文件已经被包含过，所以#ifndef后面的代码不被包含。

For the C++ language, asm is a standard keyword, but __asm__ can be used for code compiled with -fno-asm.
